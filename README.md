# Syntecxhub Weather App

A responsive, glassmorphic weather dashboard built with React and Vite for the Week 1 Web Development internship task. This application connects directly to the OpenWeatherMap API to deliver live weather observations and extended forecasts.

## Features
* **Live Weather Data**: Real-time stats including temperature, humidity, wind speed, and dynamic condition descriptions.
* **5-Day Forecast Grid**: Clean breakdown of coming conditions utilizing modular item mappings.
* **Dynamic Glassmorphic Theme**: Interface background shifts automatically based on localized weather variations (sunny, rainy, cloudy, etc.).
* **Responsive Layout**: Fluid CSS design optimized for desktop screens and mobile interfaces.

## Tech Stack
* **Frontend Framework**: React (Vite template)
* **Styling**: Pure CSS3
* **Data Provider**: OpenWeatherMap API
